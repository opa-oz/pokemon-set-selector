## How to start

Clone the repository using:

```
$  git clone --recurse-submodules https://github.com/opa-oz/pokemon-set-selector.git
```

Switch NodeJS version (for example, using NVM)
```bash
$   nvm use
```

Install dependencies:
```bash
$   yarn install
```
